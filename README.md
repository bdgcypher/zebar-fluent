A minimal and beautiful top bar following the default Windows 11 'Fluent Design' UI colorscheme.

## Features
- Numbered GlazeWM workspaces (highlights active workspace)
- Time/Date (switches when clicked)
- System information (GlazeWM tiling direction, wifi, memory usage, cpu usage, battery, and weather)
